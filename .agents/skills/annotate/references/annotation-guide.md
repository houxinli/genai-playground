# 陪读注解指南:该注什么 / 不该注什么

## 读者画像(决定密度)

读者是**日语学习者**:N4 已高分通过、**正在学 N3、目标年底考 N3**。所以:

- **注 N3 及以上**的词和语法(这是读者的学习前沿,最该帮)。
- **N4 的词也倾向于注**(还在巩固,见到有助记忆)。
- 判据:**「读者可能不认识、或读音/用法拿不准」就注。拿不准要不要注 → 注(漏注比多注差,学习者宁可多看一眼)。**
- **密度不设行级比例**——行有长有短(一行可能是一个词,也可能是一整段)。密度由**词的难度**决定:难句里注得密、简单句里注得疏,都正常。

## ✅ 该注

| 类别 | 中性例 | 注法 |
|---|---|---|
| N3+ 生词 | 詳細、募集、扱い、雰囲気、懐かしい、逞しい | `詳細(しょうさい・详细)` |
| N4 词(倾向注,巩固用) | 話題、想像、挨拶、投稿、応募、対処、招待 | `話題(わだい・话题)` |
| 音读/训读易错、生僻汉字 | 依頼(いらい)、微笑む(ほほえむ)、雫(しずく)、名残(なごり) | 读音优先 |
| N3+ 语法/句型 | 〜てはいられない、〜がる、〜きる、〜ずにはいられない、使役受身、〜わけだ | `(〜てはいられない:顾不上、没法一直…)` |
| 口语缩约/常见俚语 | 〜ちゃう(=てしまう)、〜なきゃ、めっちゃ、〜っぽい | `〜ちゃう(=てしまう:不小心/干脆做了)` |
| 活用叠加看不清原形 | 恥ずかしがって、聞かせられて | `恥ずかしがって(恥ずかしい+〜がる;て形)` |

## ❌ 不该注(注了纯噪声)

| 类别 | 例 | 为什么 |
|---|---|---|
| N5 及最基础高频词 | 人、今日、時、見る、行く、来る、思う、話す、言う、good/OK 类外来语 | N4 读者早会 |
| 拟声词/感叹词/符号 | 擬音语、あぁ、えっ、♡、〜、…… | 无固定词义,注了没用 |
| **篇内已注过的词** | 第二次及以后出现的同一词 | 只注首次 |
| 汉字+送假名一眼可读 | 見せて、行こう、着て、食べる | 直觉可读 |
| 常识性外来语 | ホテル、メッセージ、テレビ、コーヒー | 音译常识 |

## 判断顺序(拿不准时)

1. 这个词/语法是 N3 或更难吗?→ 是则注。
2. 是 N4 但读者可能拿不准读音/用法?→ 倾向注。
3. 是 N5/最基础、或非词(拟声/符号)、或本篇已注过?→ 不注。
4. 还是拿不准?→ **注**(学习者宁可多注)。

## 注解形式

- 读音+词义+语法**合一个括号**,不拆:`漢字(かんじ・词义)`、`(〜句型:简短解释)`。
- 读音覆盖括号前的**完整被注词**,包括片假名、前缀和复合词的前半部分。括号紧跟哪个词,读音就完整对应哪个词:
  - ✅ `レア度(レアど・稀有程度)`、`メートル越え(メートルごえ・超过一米)`、`濃厚乳臭(のうこうにゅうしゅう・浓重乳房气味)`
  - ❌ `レア度(ど・稀有程度)`、`メートル越え(ごえ・超过一米)`、`濃厚乳臭(にゅうしゅう・乳房气味)`
- **简短**:词义 2-6 字,语法一句话内。陪读是帮读者不卡,不是塞词典释义。
- **人名**:读音/译名以实体库(creator scope)为准;库里没有的按上下文注读音一次(首次)。
- **拿不准读音**(生僻人名/作者生造词):按最常见读法注,并在回贴里单列存疑清单。

## 原文已有括号

原文括号不是注解槽位,括号及其中内容必须逐字保留。只能在旁边插入新的括号:

- 原文:`乳内射精(なかだし)`
- ✅ `乳内射精(なかだし)(内射)`
- ❌ `乳内射精(なかだし・内射)`
