# 陪读注解指南:该注什么 / 不该注什么

读者默认 **N3**。核心判据一句话:**「N3 读者读到这里会不会卡?」卡才注,不卡不注。** 陪读是"扫读障碍",不是词典——宁可漏注简单词,不要满屏括号淹没正文。

目标密度参考:**每 2-4 个日文源行有 1 处注解**就够;一行里注解超过 3 处通常是注多了。

## ✅ 该注(N3 读者大概率会卡)

| 类别 | 例 | 注法 |
|---|---|---|
| N3+ 生词,读音/词义不直觉 | 逞しい、扱い、募集、急募、包茎、詳細 | `逞しい(たくましい・壮实)` |
| 音读易错 / 生僻汉字词 | 急募(きゅうぼ)、赦す(ゆるす)、雫(しずく) | 给读音优先 |
| 非直觉语法/句型 | 〜てはいられない、〜がる、〜きる、使役受身、〜ずにはいられない | `(〜てはいられない:顾不上、没法一直…)` |
| 口语缩约/俚语(确实卡的) | キョドる、メロつく、ピシッと、ちゃう(=てしまう) | `キョドる(畏缩、忸怩)` |
| 活用叠加看不清原形 | 恥ずかしがって、聞かせられて | `恥ずかしがって(恥ずかしい+〜がる;て形)` |

## ❌ 不该注(N3 读者不卡,注了是噪声)

| 类别 | 例(composer-2.5 实际注多了的) | 为什么不注 |
|---|---|---|
| N5/N4 高频词 | 人、女、今日、話、時、見る、行く、書く、思う | N3 早会了 |
| **常见到不该注的"看着像生词"** | **話題(わだい)、想像(そうぞう)、挨拶(あいさつ)、投稿、応募、対処** | N4/日常高频,读音也规则,**跳过** |
| 拟声词/语气词/符号 | むにゅ、ちゅぱ、あぁ、♡、〜 | 无固定词义,注了没用 |
| 篇内已注过的词 | 第二次出现的 挿乳、カリ高 等 | 只注首次 |
| 汉字+送假名一眼懂 | 見せて、行こう、着て | 直觉可读 |
| 外来语常见词 | ホテル、メッセージ、SNS、DM | 音译常识 |

## 校准:composer-2.5 首跑偏满,按此收紧

首跑把 `話題(わだい)`、`想像(そうぞう)`、`挨拶(あいさつ)`、`投稿(とうこう)`、`応募(おうぼ)` 都注了——**这些是 N4/常见词,应跳过**。收紧后同一段应该从"几乎每词都注"降到"只剩 逞しい / キョドる / メロつく / 〜てはいられない 这类真卡点"。

### 对照示例(同一句,收紧前 vs 后)

原文:
```
でも、見た目はピシっとしてんのに、結構天然だよね
```
❌ 偏满(首跑):
```
でも、見た目はピシっとしてん(ピシッと:利落干练)のに、結構(けっこう・相当)天然(てんねん・天然呆)だよね
```
✅ 收紧(結構=N4 常见,跳过;只留真卡的):
```
でも、見た目はピシっとしてん(ピシッと:利落干练)のに、結構天然(てんねん・天然呆)だよね
```

## 边界情形

- **拿不准难度**:偏向不注(漏注比满屏好)。
- **拿不准读音**(生僻人名/作者生造词):按最常见读法注,并在回贴里单列存疑清单。
- **人名**:读音/译名以实体库(creator scope)为准;库里没有的按上下文注读音一次(首次)。
- **专有名词/梗**(作品名、SNS 梗):只在 N3 读者读不懂且影响理解时,给极短说明。
