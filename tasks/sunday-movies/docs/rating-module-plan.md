# 评分模块开发计划

## 📋 项目概述

基于现有的 Fandango 场次抓取功能，开发一个综合的电影评分系统，集成多个评分源（豆瓣、IMDb、Rotten Tomatoes），为用户提供全面的电影评分信息。

## 🎯 目标

1. **多源评分集成**：整合豆瓣、IMDb、Rotten Tomatoes 的评分数据
2. **智能匹配**：根据电影标题和年份智能匹配评分数据
3. **数据聚合**：提供综合评分和详细分析
4. **用户友好**：简洁直观的评分展示界面

### 数据聚合策略
```python
# 评分权重配置
WEIGHTS = {
    'douban': 0.4,      # 豆瓣权重最高（中文用户）
    'imdb': 0.35,       # IMDb权重次之
    'rottentomatoes': 0.25  # 烂番茄权重最低
}

# 综合评分计算
def calculate_weighted_score(ratings):
    total_weight = 0
    weighted_sum = 0
    
    for rating in ratings:
        weight = WEIGHTS.get(rating.source, 0)
        normalized_score = rating.score / rating.scale
        weighted_sum += normalized_score * weight
        total_weight += weight
    
    return weighted_sum / total_weight if total_weight > 0 else 0
```
